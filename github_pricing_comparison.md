## Available Add-ons

### For All Plans
- **GitHub Copilot**: Free tier with 2,000 completions and 50 chat requests/month
- **GitHub Codespaces**: Cloud dev environments, compute from $0.18/hr, storage $0.07/GB/month

### For Team and Enterprise
- **GitHub Secret Protection**: Push protection, secret scanning, custom patterns
- **GitHub Code Security**: Vulnerability detection, Copilot Autofix, CodeQL

### Enterprise Exclusive
- **Premium Support**: 30-minute SLA on Urgent tickets, 24/7 web and phone support
- **Premium Plus Support**: Includes assigned Customer Reliability Engineer
- **GitHub Models**: Industry-leading AI in workflow
- **GitHub Advanced Security**: Comprehensive security suite

## Key Differentiators by Tier

### Free → Team Upgrades
- Web-based support (vs community only)
- Full repository rules for private repos
- Code owners for private repos
- Draft pull requests for private repos
- Required reviewers feature
- Multiple reviewers in pull requests
- Environment deployment branches and secrets
- Pages and wikis for private repos
- Audit log access
- 50% more CI/CD minutes (3,000 vs 2,000)
- 4x more package storage (2 GB vs 500 MB)

### Team → Enterprise Upgrades
- 16x more CI/CD minutes (50,000 vs 3,000)
- 25x more package storage (50 GB vs 2 GB)
- SAML SSO
- Audit log API
- Enterprise Managed Users
- SCIM user provisioning
- Data residency options
- GitHub Connect
- SOC1/SOC2 compliance reports
- FedRAMP authorization
- IP allow list
- Environment protection rules for all repos
- Invoice billing
- Premium support availability
- Self-hosted deployment option (Enterprise Server)
